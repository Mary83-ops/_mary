Weather App

The Weather App is a web application designed to display real-time weather information for any location. Developed using vanilla JavaScript and the OpenWeatherMap API, the app allows users to enter a location and receive weather information such as temperature, humidity, wind speed, and more. The app's dynamic design ensures that the weather information is displayed in a clear and organized way, with icons and visuals to enhance user experience. The OpenWeatherMap API was used to retrieve and display weather data, while vanilla JavaScript was used to manipulate the DOM and create the app's interactive elements. Overall, the project showcases proficiency in web development and the ability to create user-friendly applications using popular technologies and APIs.
